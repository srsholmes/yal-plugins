## @yal-app/plugin-stripe-test-cards

This is a plugin for Yal which displays a list of Stripe test cards. Select the card you want to copy and it will be copied to your clipboard.

## Screenshot

![stripe-test-cards](./resources/stripe-test-cards.png 'stripe-test-cards')

## How do I install it?

From the root directory of this repo run:

```
yarn install:plugin stripe-test-cards
```
